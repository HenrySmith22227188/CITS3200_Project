Multiple Goal Pursuit Game

How to Play:
  • The Goal is to win as many points as possible across all of the trials
  • In each trial, you get the points by having the most cards in a row
  • your scores are different based on the amount of points indicated at the end of a row
  • Each row has a different number of points to be won indicated on the left of each lane the opponent may have a different value indicated on the right
  • Each turn you can place one card in a row until you run out of cards
    • Basic Rules:
  • You can only place one card per turn
  • Both players must decide where to place a card before The other player's decision is shown
  • Each player has a set number of cards, configured at the beginning by an admin
  • Trials end when both players have no cards left
