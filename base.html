How to Play:

This is a two-player game. The aim of the game is to win as many points as possible in multiple game-trials.

In each game-trial:
  • Each row on the screen represents a 'goal'.
  • Each 'goal' is worth a set of points (indicated on the left for player-1 and on the right for player-2).
  • Each 'card' (at the bottom of the screen) holds a bidding-value shown on the card.
  • Over multiple rounds, players place a single 'card' into a row to bid on a 'goal'.
  • The numbers in the centre indicate the player's current total bid on a ‘goal’.
  • After the last round, the player with the highest bid on a ‘goal’ wins the points specified on it.

    • The player with the highest points at the end wins the game.

    Basic Rules:

  • Each player has a set number of cards, configured at the beginning by the admin.
  • A players must only place one card per round/turn.
  • The other player's decision is only shown when both players end their turn.
  • A trial will end when both players have no cards left.
